 # Loan Approval Prediction System

# Project Description
The Loan Approval Prediction System is a Machine Learning project that predicts whether a loan application will be approved based on applicant details. It uses a Logistic Regression model to analyze the input data and predict the loan approval status.

# Project Objective
To simplify the loan approval process by predicting loan eligibility using Machine Learning.

# Features
- Predicts loan approval status
- User-friendly web interface
- Machine Learning model using Logistic Regression
- Fast and accurate prediction

## How to Run
1. Clone the repository.
2. Install the required libraries using:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open the application in your browser.
