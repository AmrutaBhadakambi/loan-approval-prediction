# Loan Approval Prediction System

## Project Description
The Loan Approval Prediction System is a Machine Learning project that predicts whether a loan application will be approved based on applicant details.

## Project Objective
To simplify the loan approval process by predicting loan eligibility using Machine Learning.

## Features
- Predicts loan approval status
- User-friendly web interface
- Machine Learning model using Logistic Regression
- Fast and accurate prediction

## Technologies Used

- Flask
- HTML
- CSS
- JavaScript
- Pandas

## How to Run
1. Clone the repository.
2. Install the required libraries.
3. Run `app.py`.
4. Open the application in your browser.

## Output
The system predicts whether the loan application is **Approved** or **Rejected** based on the user's input.
